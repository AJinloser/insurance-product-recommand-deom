# 保险产品推荐方法论

> 本文档面向 LLM Prompt 设计，梳理经纪人推荐保险产品的思考框架，帮助模型更准确地理解产品特征、匹配客户需求。



## 一、三类医疗险的产品特性与推荐要点

### 1.1 百万医疗险

**产品定位**：高保额、低保费，主要覆盖大病住院的灾难性医疗支出。

**核心特征**
- 保额通常 100 万～600 万，年保费几百元
- 免赔额较高（通常 1 万元），适合有社保打底的人群
- 主要覆盖公立医院普通部，部分产品含特需/国际部
- 健康告知较严，既往症通常除外承保
- 续保稳定性是核心风险点（非保证续保产品停售风险）

**推荐时重点关注**
- 续保条件：是否保证续保？停售后是否可转保？
- 免赔额设计：是否与社保报销后的自付部分衔接合理
- 健康告知：客户是否有可能触发除外条款
- 是否含质子重离子等高端治疗

**适合人群**：有社保、身体健康、预算有限、主要防范大病风险的年轻/中年人群。

---

### 1.2 中端医疗险

**产品定位**：介于百万医疗和高端医疗之间，覆盖更广的就医场景，兼顾性价比与服务质量。

**核心特征**
- 保额通常 50 万～200 万，年保费数千元
- 免赔额较低或为零，报销比例较高
- 可覆盖公立医院特需部、部分私立医院
- 部分产品含门诊责任
- 健康告知相对灵活，部分产品支持智能核保

**推荐时重点关注**
- 医院网络：覆盖哪些私立/特需医院，是否匹配客户所在城市
- 门诊是否包含：门诊责任对保费影响大，需与客户确认是否必要
- 直付 vs 报销：直付医院范围是否实用
- 与百万医疗的差异化价值：客户是否真的需要升级（避免过度销售）

**适合人群**：有一定预算、希望就医体验更好（减少排队、选择更多医院）、对门诊有需求的中产人群。

---

### 1.3 高端医疗险

**产品定位**：全面覆盖、高服务标准，面向高净值人群或有国际就医需求的客户。

**核心特征**
- 保额通常 500 万以上甚至无上限，年保费数万元
- 覆盖国内外高端私立医院、国际医院
- 通常含门诊、齿科、视光、生育等全面责任
- 直付网络广泛，理赔服务体验好
- 健康核保相对宽松（部分产品），但保费随健康状况浮动明显

**推荐时重点关注**
- 境外保障范围：是否含美国（保费差异极大）
- 齿科/生育等附加责任：是否真正需要，避免为不需要的责任付费
- 保险公司偿付能力与服务口碑：高端险更依赖服务兑现
- 与雇主团险的关系：是否已有雇主提供高端医疗，避免重复
- 费率增长预期：高端险保费随年龄增长显著，需做长期规划

**适合人群**：高净值人群、外籍人士、有海外就医需求、追求就医体验无妥协的客户。

## 二、跨产品类型的选择逻辑

当客户在不同类型医疗险之间犹豫时，可用以下决策树辅助判断：

```
客户预算是否充足（年保费 > 1 万）？
├── 否 → 百万医疗险（优先保证续保版本）
└── 是 → 是否有私立/国际医院就医需求？
         ├── 否 → 中端医疗险（性价比优先）
         └── 是 → 是否有境外就医或极致服务需求？
                  ├── 否 → 中端医疗险（含私立网络）
                  └── 是 → 高端医疗险
```

> **LLM 提示**：以上决策树是简化模型，实际推荐中需结合健康状况、家庭结构、已有保障等因素综合判断，不可机械套用。

## 三、常见推荐误区

| 误区 | 正确做法 |
|------|----------|
| 只比较保额大小 | 结合免赔额、报销比例、医院网络综合评估实际保障价值 |
| 忽视续保稳定性 | 续保条件是医疗险最重要的长期风险，必须明确说明 |
| 用通用描述代替个性化推荐 | 推荐理由必须与客户具体情况挂钩 |
| 忽略健康告知风险 | 有既往症的客户，必须评估核保结果再推荐 |
| 过度推荐高保费产品 | 在客户预算范围内找最优解，而非默认推荐最贵的 |
| 忽视已有保障 | 先了解社保和已购商业险，避免重复保障 |
